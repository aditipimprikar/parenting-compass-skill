# Example Generated Skills

This folder contains real examples of parenting compass skills generated by the `/create-parenting-skill` tool.

## How to Use These Examples

**These are NOT meant to be copied directly.** Each skill is deeply personalized to a specific parent's values, child's age, and challenges.

**Instead, use them to:**
- See what the generator creates
- Understand the structure and format
- Get inspired by different approaches
- Decide if this tool is right for you

**To create YOUR OWN skill:** Run `/create-parenting-skill` and answer the questions based on YOUR family's needs.

## Example 1: Whole-Brain Approach (Toddler, 19 months)

**File:** [whole-brain-toddler-example.md](whole-brain-toddler-example.md)

**Parent's Context:**
- Child: 19 months old
- Challenges: Meltdowns, discipline/boundaries, building independence
- Values: Connection before correction, brain development over compliance, independence building
- Practices: No screens (except grandparent calls), outdoor time, multilingual exposure, involvement in daily tasks

**Approach:** 
- Brain science-based (The Whole-Brain Child, Raising Good Humans, Let Grow)
- PAUSE framework for decision-making
- Emphasis on connection, natural consequences, and age-appropriate autonomy
- Reflective listening and repair practices

**Best for parents who:**
- Want brain science backing their approach
- Value both connection AND independence
- Are breaking generational patterns
- Focus on long-term development over short-term compliance

## Want to Share Your Generated Skill?

If you've used the generator and created a skill that might help others, consider:

1. **Anonymize it** - Remove personal details, child's name, specific family situations
2. **Add context** - Explain your parenting approach and what you optimized for
3. **Submit a PR** - Share it as an example in this folder

The more examples we have showing different approaches (attachment, Montessori, traditional, etc.), the more people understand what the generator can create!
